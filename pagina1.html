Glosario de TI (A-M)

A B C D E F G H I J K L M
A

API: Interfaz de Programación de Aplicaciones.

Algoritmo: Conjunto de instrucciones para resolver un problema.

Autenticación: Proceso de verificar la identidad de un usuario.

Archivo: Unidad de almacenamiento de datos.

Aplicación: Programa diseñado para realizar tareas específicas.

B

Backend: Parte del desarrollo web que gestiona la lógica del servidor.

Base de datos: Conjunto organizado de datos almacenados electrónicamente.

Bug: Error en un programa de software.

Big Data: Conjunto de datos masivos y complejos.

Blockchain: Tecnología de registro distribuido.

C

Cache: Almacenamiento temporal para acelerar el acceso a datos.

Cloud Computing: Uso de recursos informáticos a través de internet.

Compilador: Programa que traduce código fuente a código máquina.

Criptografía: Técnica para proteger información mediante cifrado.

CPU: Unidad Central de Procesamiento.

D

DevOps: Metodología que combina desarrollo y operaciones.

DNS: Sistema de nombres de dominio.

Data Mining: Proceso de analizar grandes conjuntos de datos.

Driver: Software que permite la comunicación con hardware.

Debugging: Proceso de identificar y corregir errores en el código.

E

Ethernet: Tecnología de red cableada.

Encriptación: Proceso de codificar datos para protegerlos.

Endpoint: Punto final de comunicación en una red.

Escalabilidad: Capacidad de un sistema para crecer según la demanda.

ERP: Sistema de planificación de recursos empresariales.

F

Firewall: Sistema de seguridad para redes.

Frontend: Parte visible de una aplicación web.

Framework: Conjunto de herramientas para desarrollo.

FTP: Protocolo de transferencia de archivos.

Función: Bloque de código reutilizable.

G

Gateway: Puerta de enlace en redes.

Git: Sistema de control de versiones.

GPU: Unidad de procesamiento gráfico.

Gigabyte: Unidad de almacenamiento de datos.

GraphQL: Lenguaje de consulta para APIs.

H

HTML: Lenguaje de marcado para la web.

HTTP: Protocolo de transferencia de hipertexto.

Hash: Valor único generado por un algoritmo.

Hosting: Servicio de alojamiento web.

Hypervisor: Software para virtualización.

I

IP: Protocolo de internet.

IoT: Internet de las cosas.

IDE: Entorno de desarrollo integrado.

IPv6: Versión 6 del protocolo IP.

Instancia: Ejecución de un objeto en programación.

J

Java: Lenguaje de programación.

JavaScript: Lenguaje de programación para la web.

JSON: Formato de intercambio de datos.

JVM: Máquina virtual de Java.

Job: Tarea o proceso en un sistema.

K

Kubernetes: Plataforma de orquestación de contenedores.

Kernel: Núcleo de un sistema operativo.

Key: Clave utilizada en criptografía.

Keyword: Palabra clave en programación.

Kibibyte: Unidad de almacenamiento.

L

Linux: Sistema operativo de código abierto.

Log: Registro de eventos en un sistema.

Load Balancer: Balanceador de carga.

Library: Conjunto de funciones reutilizables.

Latency: Tiempo de retraso en una red.

M

Middleware: Software que actúa como puente entre aplicaciones.

Machine Learning: Rama de la inteligencia artificial.

Microservicios: Arquitectura basada en servicios pequeños.

Monitorización: Supervisión de sistemas.

Memoria RAM: Memoria de acceso aleatorio.
