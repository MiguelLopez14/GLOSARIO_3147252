Glosario de TI (N-Z)

N Ñ O P Q R S T U V W X Y Z
N

Network: Red de computadoras interconectadas.

Node: Punto de conexión dentro de una red.

Namespace: Espacio de nombres para organizar elementos.

Null: Valor vacío o inexistente en programación.

NIC: Tarjeta de interfaz de red.

Ñ

Ñu: Nombre del proyecto GNU en español.

Ñandú: Ejemplo de palabra con Ñ (sin uso técnico).

Ñapa: Término usado en algunos sistemas para "extra".

Ñuño: Ejemplo ficticio para completar la letra Ñ.

Ñame: Ejemplo de palabra con Ñ (sin uso técnico).

O

Open Source: Software de código abierto.

Object: Entidad en programación orientada a objetos.

Operating System: Sistema operativo.

Optimization: Proceso de mejora de rendimiento.

Overflow: Error por exceder un límite de memoria.

P

Protocol: Conjunto de reglas para comunicación.

Programming: Proceso de escribir código.

Packet: Unidad de datos en redes.

Port: Punto de conexión en redes.

Python: Lenguaje de programación popular.

Q

Query: Solicitud de datos en una base de datos.

Queue: Estructura de datos tipo cola.

QoS: Calidad de servicio en redes.

Quantum Computing: Computación cuántica.

QuickSort: Algoritmo de ordenamiento rápido.

R

Router: Dispositivo que dirige el tráfico de red.

RAM: Memoria de acceso aleatorio.

Repository: Almacén de código fuente.

REST: Estilo arquitectónico para servicios web.

Runtime: Tiempo de ejecución de un programa.

S

Server: Computadora que proporciona servicios a otras.

SQL: Lenguaje de consulta estructurado.

SSH: Protocolo para acceso remoto seguro.

Stack: Estructura de datos tipo pila.

Script: Archivo con instrucciones ejecutables.

T

TCP: Protocolo de control de transmisión.

Thread: Hilo de ejecución en un programa.

Token: Unidad de autenticación.

Topology: Disposición de una red.

Transaction: Operación en una base de datos.

U

URL: Localizador uniforme de recursos.

User: Persona que interactúa con un sistema.

UDP: Protocolo de datagramas de usuario.

Unicode: Estándar de codificación de caracteres.

Uptime: Tiempo que un sistema está operativo.

V

Virtualization: Creación de recursos virtuales.

VPN: Red privada virtual.

Versioning: Control de versiones de software.

Variable: Espacio en memoria para almacenar datos.

VLAN: Red de área local virtual.

W

Web: Sistema de información en internet.

WiFi: Tecnología de red inalámbrica.

WAN: Red de área amplia.

Widget: Elemento gráfico en una interfaz.

Workflow: Flujo de trabajo.

X

XML: Lenguaje de marcado extensible.

XSS: Vulnerabilidad de scripts entre sitios.

XOR: Operador lógico exclusivo.

Xamarin: Plataforma para desarrollo móvil.

X86: Arquitectura de procesadores.

Y

YAML: Lenguaje de serialización de datos.

Yottabyte: Unidad de almacenamiento masivo.

Yield: Declaración en programación para generar valores.

YUI: Biblioteca de interfaz de usuario.

Yarn: Gestor de paquetes para JavaScript.

Z

Zero-Day: Vulnerabilidad sin parche conocido.

Zip: Formato de compresión de archivos.

Zettabyte: Unidad de almacenamiento masivo.

Zigbee: Protocolo de comunicación inalámbrica.

Zone: Segmento de red o espacio definido.
